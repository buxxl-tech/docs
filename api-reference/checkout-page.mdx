### 功能描述

收银台页面的核心功能包括：
- **一站式收款服务**：为商户提供标准化的存款页面，减少开发复杂度。
- **信息展示**：显示以下与存款相关的关键信息：
  - 存款地址
  - 链类型
  - 币种
  - 存款金额
  - 过期时间倒计时
  - 订单号
- **用户友好性**：支持二维码展示，用户可直接扫码完成存款。
- **安全性**：通过 URL 编码隐藏所有关键信息，避免明文展示。

---

### 商户使用场景

收银台页面适用于以下场景：
1. **APP 或 WEB 端嵌入收银台页面**：
   - 商户可以在其应用中嵌入收银台页面，用户无需跳转即可完成存款。
2. **线下收款码展示**：
   - 商户可以在实体店或线下场景中展示包含收银台信息的二维码，用户扫描后可通过页面完成存款。

### 收银台页面的核心优势

#### **1. 用户体验优化**
- 提供友好的交互设计，支持二维码扫码支付。
- 信息展示清晰，用户能够快速完成支付操作。

#### **2. 安全保障**
- 参数通过 URL 编码隐藏，所有敏感信息不以明文形式展示。
- 支持过期时间限制，超时后无法继续充值，避免错误支付。

#### **3. 易于集成**
- 商户无需额外开发，只需嵌入收银台页面，即可快速上线存款功能。

### 收银台页面的可选信息

收银台页面支持以下可选信息展示，商户可根据需求选择是否启用：
- **存币须知**：例如链类型、是否支持小额充值等说明。
- **汇率信息**：动态展示当前币种的汇率，便于用户了解换算金额。
- **商户 LOGO**：自定义底部品牌标识，增强品牌感知。
- **提示信息**：例如“请勿充值非支持币种，否则资产将无法找回”等安全提示。

### 总结

收银台页面为商户提供了标准化的存款流程和安全保障，适用于多场景的支付需求。通过嵌入此页面，商户可以快速为客户提供高效、安全的区块链存款服务。
